# Get started with Foton

<Callout type="warning" emoji="⚠️">
  Be careful, Foton is still in the early stages of development and may have breaking changes in the future.
</Callout>

Foton is a TypeScript toolkit for interacting with TON wallets and blockchain as a whole. The library wraps existing solutions into one comfortable API. It supports:

- ✅ Wallet connection
- ✅ Sending transactions
- ✅ Querying the blockchain
- ✅ Deploying and interacting with smart contracts

Try creating a simple Counter dApp with React and Foton by following [the tutorial](./docs/guides/tutorial.mdx).
